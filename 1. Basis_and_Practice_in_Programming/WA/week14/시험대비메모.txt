***입력 관련 함수***
scanf("~~~~", &~~) : 문자 받아줌. 스페이스, 탭 엔터 전까지
getchar() : 한번에 문자 하나 받아줌
fgets(받아서 저장해둘 변수, 사이즈, FILE *stream) : 한 줄 받아줌
fgetc(파일 포인터) : 파일로부터 하나의 문자를 받아옴
fscanf(파일 포인터, "~~", ...) : 파일 버전 scanf
fgets(저장할 변수, 문자열 길이, 파일 포인터) : 한 줄을 읽어줌

***출력 관련 함수***
printf("~~~~", 변수) : 원하는대로 출력
puts("~~~~", 변수) : printf와 동일. 대신 마지막에 엔터가 자동 탑재되어있음
fprintf(파일 포인터, "~~", ...) : 파일 버전 printf

***기타 함수***
strcmp(char 1, char 2) : 두 변수를 비교해줌. 완전히 같을 경우에 0을 리턴해줌
strlen(string변수) : 변수의 길이를 리턴해줌
is~~(변수) : ~~인지 판단해주는함수. 예를들어 알파벳. 거짓이면 0을 리턴한다
toupper(변수) : 대문자로 바꿔줌
tolower(변수) : 소문자로 바꿔줌
